SAÉ 1-01 : Reporting à partir d’un SGBD

Présentation du projet

Dans ce projet nous avons travaillé sur une base de données nommée « victimes_police_US.db », qui recense les décès survenus lors d’interventions policières aux États-Unis. Cette base contient des informations détaillées sur les victimes, permettant d’extraire des statistiques précises et pertinentes.

Démarche

1. Exploration : Analyse de la base de données « victimes_police_US.db » avec SQLite et Python pour identifier les variables clés.
2. Extraction : Utilisation de requêtes SQL et de scripts Python pour calculer des statistiques (répartition démographique, géographique, etc.).
3. Visualisation : Création de tableaux et graphiques synthétiques pour illustrer les résultats.
4. Export : Génération d’un fichier Excel structuré avec les données et visualisations.
5. Validation : Vérification des résultats et documentation pour garantir leur fiabilité.

Acquis

  • Maîtrise de l'exploration de bases de données avec SQLite et Python.
  • Capacité à extraire et analyser des données à l'aide de requêtes SQL et de scripts Python (pandas, numpy).
  • Compétence en visualisation de données avec des outils comme matplotlib et seaborn.
  • Savoir-faire dans la création de rapports structurés avec des tableaux et graphiques synthétiques.
  • Expérience dans l'export et la mise en forme de données vers des fichiers Excel.
  • Renforcement des compétences en validation et documentation pour garantir la fiabilité des résultats.
  • Capacité à automatiser des processus d'analyse pour gagner en efficacité.
  • Compréhension des enjeux sociétaux liés aux données analysées.
